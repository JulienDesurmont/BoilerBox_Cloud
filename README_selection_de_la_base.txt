Cet ajout a pour but de pouvoir switcher entre les bases de données depuis le même projet BoilerBox

Prérequis pour switcher entre les bases de données
Liste déroulante indiquant la liste de sites disponibles

Modification de la variable contenue dans le fichier /etc/environnement

Modification des fichiers de configurations suivants pour l'accès à la base de données : 
   - app/config/parameters.yml
   - web/config_ipc
